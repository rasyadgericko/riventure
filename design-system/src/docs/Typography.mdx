# Typography

RYC uses two typefaces: **Cabinet Grotesk** for display headings and **Inter** for body text. Both are self-hosted as WOFF2 for performance.

---

<HumanOnly>

## Heading Hierarchy

All headings use Cabinet Grotesk in uppercase with tight tracking. The scale is fluid — sizes respond to viewport width via `clamp()`.

<HeadingPreview />

## Type Scale

Every heading and body style in the system, rendered at actual size.

<TypeScale tokens={typeScale} />

## Font Families

| Role | Typeface | Weights | Usage |
|------|----------|---------|-------|
| Display | Cabinet Grotesk | 700, 900 | Hero headings, section titles |
| Body | Inter | 400, 500, 600 | Body text, labels, UI elements |
| Mono | System mono stack | 400 | Code snippets, token values |

### Guidelines

- **Headings** always use Cabinet Grotesk in uppercase with tight letter-spacing (`-0.05em`).
- **Body text** uses Inter at `0.95rem` with relaxed line-height (`1.8`) for readability.
- **Tags and eyebrows** use Inter at `0.7rem`, semibold, uppercase, with `0.08em` tracking.
- Use `<em>` within headings for italic emphasis — this is a core brand pattern.

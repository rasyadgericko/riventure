# Shadows & Radii

Elevation and rounding tokens for layered interfaces. Shadows use `color-mix()` with `var(--fg)` so they adapt automatically to the active theme.

### Guidelines

- Cards use `--radius-md` (8px). Testimonial cards use `--radius-lg` (12px).
- Buttons and tags always use `--radius-pill` (100px) for the pill shape.
- Avatars and circular icons use `--radius-round` (50%).
- Shadows scale with interaction: resting = `--shadow-sm`, hover = `--shadow-lg`.

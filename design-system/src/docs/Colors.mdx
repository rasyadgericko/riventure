# Colors

RYC uses a monochromatic palette that inverts cleanly between light and dark themes. Every color is defined as a CSS custom property and adapts automatically via `data-theme`.

### Usage Guidelines

- **Never hardcode hex values** — always use `var(--token-name)`.
- `--fg` and `--bg` carry the theme. Everything else derives from these.
- Use `--fg-muted` for secondary text, `--fg-dim` for decorative/tertiary.
- `--border` uses alpha transparency so it works on any background.
- Semantic colors (`--success`, `--error`, etc.) are theme-agnostic — same value in both modes.

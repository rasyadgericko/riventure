# Spacing

RYC uses a **4px base grid**. Every margin, padding, and gap snaps to this scale for visual consistency across components and layouts.

---

## Scale

<SpacingScale tokens={spacingScale} />

<HumanOnly>

### Layout Patterns

| Context | Token | Usage |
|---------|-------|-------|
| Section padding (desktop) | `--space-20` + `4vw` | Vertical + horizontal section spacing |
| Section padding (mobile) | `--space-16` + `6vw` | Reduced on smaller screens |
| Card padding | `--space-10` – `--space-12` | Internal card content spacing |
| Card grid gap | `--space-3` | Space between card grid items |
| Form field gap | `--space-6` | Vertical gap between form fields |
| Two-column layout gap | `--space-12` – `--space-16` | About section, contact split |
| Button padding | `--space-5` (v) × `--space-10` (h) | Primary button internal spacing |
| Tag padding | `--space-1` (v) × `--space-3` (h) | Pill-shaped tags and badges |

### Guidelines

- Always use spacing tokens instead of arbitrary pixel values.
- For responsive adjustments, drop one or two steps rather than halving values.
- `--space-3` (12px) is the default gap for dense grids; `--space-6` (24px) for relaxed layouts.

</HumanOnly>
